CentOS Install
CentOS 5, Interactive Installation (paravirtualized kernel)

Type: ConVirt Provisionable Image
Version: 1.0
Author: The ConVirt Team
Requirements: ConVirt v0.8+, Xen v3.0.4+, 32-bit, Xen-enabled Linux host.
Description: This image creates a 5G virtual disk (vbd) and kicks off an interactive, anaconda installation for CentOS 5. The kernel_src and ramdisk_src point to the location from where the initial kernel and ramdisk are to be downloaded.
Instructions:
  1. Check the default provisioning parameters and alter them to suit your environment if required.
  2. Provision the VM configuration normally using ConVirt.
  3. Start the Virtual Machine (VM) and complete the installation.
  4. (Important) Once the installation is complete, edit the new VM's configuration and change the kernel and ramdisk entries to EITHER point to a local copy of a xen paravirtualised kernel and associated ramdisk. OR set the bootloader entry (typically /usr/bin/pygrub)
  5. Restart the VM.
