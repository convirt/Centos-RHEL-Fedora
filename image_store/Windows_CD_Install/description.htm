Windows CD Install
Windows Installation from CD/DVD (unmodified)

Type: ConVirt Provisionable Image
Version: 1.0
Author: The ConVirt Team
Requirements: ConVirt v0.8+, Xen v3.0.4+ OR KVM-70+, Hardware Virtualization (Intel VT/AMD SVM) enabled processor, Xen-enabled Linux host, Bootable Windows Installation CD/DVD.
Description: This image creates a 10G virtual disk (vbd) for use as the primary hard drive. The VM boots from a bootable Windows installation CD/DVD which, in turn, should kick off the interactive Windows installation routine and deploy the OS on the primary hard drive.
Instructions:
  1. Check the default provisioning parameters and alter them to suit your environment if necessary.
  2. (Important) The default location for the CD/DVD drive is set to /dev/cdrom. Alter this to match your environment if necessary.
  3. Provision the VM configuration normally using ConVirt.
  4. Start the Virtual Machine (VM) and complete the installation.
  5. (Important) Once the installation is complete, use the "Set boot Device" context menu to change the boot device to the Disk. (This ensures that the VM boots from disk as opposed to CD on the next restart)
  6. Restart the VM.
