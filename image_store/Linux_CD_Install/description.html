Linux CD Install
Linux Installation from CD/DVD (unmodified)

Type: ConVirt Provisionable Template

Version: 2.0

Author: The ConVirt Team

Requirements: ConVirt v0.8+, Xen v3.0.4+ OR KVM-70+, Hardware Virtualization (Intel VT/AMD SVM) enabled processor, Xen-enabled Linux host, Bootable Linux CD/DVD.

Description: This template creates a 10G virtual disk (vbd) for use as the primary hard drive. The VM boots from a bootable Linux CD/DVD which, in turn, should kick off the distribution specific installation routine and deploy the OS on the primary hard drive.

Instructions:
1. Check the default provisioning parameters and alter them to suit your environment if necessary
2. (Important) The default location for the CD/DVD drive is set to /dev/cdrom. Alter this to match your environment if necessary.
3. Provision the VM configuration normally using ConVirt.
4. (Important) Once the installation is complete, use the "Set boot Device" context menu to change the boot device to the Disk. (This ensures that the VM boots from disk as opposed to CD on the next restart)
5. Restart the VM.
